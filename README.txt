=== ARC Raiders Screen Filter ===

Экранный фильтр для осветления тёмных участков в ARC Raiders.
Работает через Windows API — не является читом, не инжектится в игру.

---

УСТАНОВКА:

1. Запустить enable_extended_gamma.reg (2 клика, разрешить изменения)
2. ПЕРЕЗАГРУЗИТЬ ПК
3. Запустить start.bat от администратора (правый клик → Запуск от имени администратора)

---

УПРАВЛЕНИЕ:

B   — переключить пресет (off → medium → strong → off)
N   — выключить фильтр (вернуть стандартные настройки)
F12 — выход из скрипта

---

ПРЕСЕТЫ:

off    — стандартные настройки Windows
medium — среднее осветление (подходит для большинства локаций)
strong — сильное осветление (тёмных мест практически нет)

---

ВАЖНО:

• Скрипт должен работать в фоне во время игры
• При выходе/крэше гамма автоматически восстанавливается
• Фильтр влияет на весь экран, не только на игру
