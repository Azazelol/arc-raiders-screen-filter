# ARC Raiders Screen Filter

> **Экранный фильтр для осветления тёмных участков в ARC Raiders**

Работает через Windows API `SetDeviceGammaRamp` — не затрагивает процесс игры, не инжектит DLL, **не является читом**.

---

## 🎯 Зачем это нужно?

ARC Raiders использует очень тёмную цветовую палитру в некоторых локациях. Этот скрипт поднимает яркость тёмных участков экрана, оставляя яркие относительно нетронутыми — таким образом вы видите врагов в тени, не теряя детализацию.

**Почему не NVIDIA Freestyle?** — Разработчики заблокировали его для ARC Raiders.

**Почему не ReShade?** — Может быть заблокирован античитом (DLL инжекция).

**Этот скрипт** работает на уровне ОС и технически невозможно заблокировать.

## 📥 Установка

### Требования
- Windows 10/11
- Python 3.12+ (скрипт установит автоматически, если запустить через `start.bat`)
- Права администратора

### Шаги

1. **Скачать репозиторий**
   ```bash
   git clone https://github.com/Azazelol/arc-raiders-screen-filter.git
   cd arc-raiders-screen-filter
   ```
   Или [скачать ZIP](https://github.com/Azazelol/arc-raiders-screen-filter/archive/refs/heads/master.zip) и распаковать.

2. **Установить расширенный диапазон гаммы**

   Запустить `enable_extended_gamma.reg` (двойной клик, разрешить изменения)

   > Этот файл добавляет ключ реестра `HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ICM\GdiICMGammaRange = 256`, который снимает ограничение Windows на изменение гамма-кривой.

3. **Перезагрузить ПК**

4. **Запустить скрипт**

   Правый клик по `start.bat` → **Запуск от имени администратора**

## ⚙️ Пресеты

| Пресет | brightness | contrast | gamma | Описание |
|--------|-----------|----------|-------|----------|
| **off** | 1.0 | 1.0 | 1.0 | Стандартные настройки Windows |
| **medium** | 1.2 | 1.2 | 0.5 | Среднее осветление — подходит для большинства локаций |
| **strong** | 1.3 | 1.3 | 0.35 | Сильное осветление — тёмных мест практически нет |

> **Ключевой параметр — gamma < 1.0**. Именно он поднимает тёмные участки, оставляя яркие относительно нетронутыми.

## 🛡️ Безопасность

### Это безопасно?
✅ **Да.** Скрипт работает на уровне ОС через стандартный Windows API.

### Это читерство?
❌ **Нет.** Скрипт не модифицирует:
- Процесс игры
- Файлы игры
- Память игры
- Сетевой трафик

Он просто меняет гамма-кривую монитора — аналогично тому, как вы бы изменили настройки яркости в меню монитора.

### Можно ли получить бан?
🤷 **Маловероятно.** Скрипт не взаимодействует с игрой напрямую. Это аналогично использованию настроек монитора или драйвера видеокарты.

## 🔧 Как это работает?

```python
# Получаем Device Context монитора
hdc = user32.GetDC(0)

# Строим кастомную гамма-кривую
for i in range(256):
    value = pow(i / 255.0, gamma)  # gamma < 1.0 поднимает тени
    value = (value - 0.5) * contrast + 0.5
    value *= brightness
    ramp[channel][i] = int(value * 65535)

# Применяем к монитору
gdi32.SetDeviceGammaRamp(hdc, ramp)
```

При выходе из скрипта (включая крэш) гамма автоматически восстанавливается через `atexit`.

## 🐛 Проблемы и решения

### Скрипт не меняет яркость

**Решение:**
1. Убедитесь, что запущен **от администратора**
2. Проверьте, что выполнили `enable_extended_gamma.reg` и **перезагрузили ПК**
3. Выключите HDR в настройках Windows (Settings → Display → HDR)
4. Проверьте вывод `[TEST]` в консоли при запуске

### Клавиши не работают во время игры

**Причина:** Библиотека `keyboard` требует прав администратора для глобальных хоткеев.

**Решение:** Запускайте `start.bat` от администратора.

### Python not found

**Решение:** Закройте и откройте консоль заново, или установите Python вручную с [python.org](https://www.python.org/downloads/)
